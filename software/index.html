xswap

Xswap is a Python package with a C/C++ extension for degree-preserving network permutation. Permuted networks allow counterfactual distributions to be computed for features used in many network analysis or prediction methods, especially link prediction. Hetnets can be permuted by metaedges, and permuted hetnets allow heterogeneous network link prediction features to be compared to a meaningful counterfactual distribution produced by computing the same features on permuted networks. The C/C++ implementation allows for very rapid network permutation.
