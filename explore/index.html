Neo4j Browser

Neo4j is a third-party graph database that supports the Cypher language for querying and visualizing hetnets. Users can make advanced queries on Hetionet right away – without downloading anything – by viewing using the public Neo4j browser app hosted here. The Neo4j instance also provides API access, which allows services to be built on top of Hetionet, like connectivity search above.

Download

Hetionet is available in multiple machine-readable formats.
  • JSON .json (readable by hetnetpy)
  • tablular .tsv (alternative when JSON not possible/practical)
  • Neo4j .graph (for import into Neo4j)
  • matrix .npy .npz (readable by hetmatpy)
Users looking to perform more advanced queries and analyses not supported by the web apps above can use these datasets for local processing.
